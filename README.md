# Kana & Kanji Random Practice

A lightweight Japanese character practice web app for learning:

- Hiragana
- Katakana
- Jōyō Kanji (Elementary + Secondary school level)

Designed for quick repetition-based study with a distraction-free interface on both desktop and mobile.

---

## Features

### Random Character Practice

Click anywhere on the screen to generate a random character.

Supported modes:

- Hiragana
- Katakana
- Kanji

Modes can be enabled simultaneously for mixed practice.

---

### Instant Answer Reveal

Press the `정답` button to reveal:

- Romaji pronunciation for Kana
- Reading and meaning for Kanji

---

### Kanji Practice Filters

Kanji mode supports filtering by Japanese school level:

- Elementary school Kanji (1,006 characters)
- Secondary school Kanji (1,130 characters)

You can:

- Practice only elementary Kanji
- Practice only secondary Kanji
- Mix both together

---

### Kana & Kanji Tables

Top-right buttons open reference tables:

- `あ` → Hiragana table
- `ア` → Katakana table
- `漢` → Kanji reference list

Kanji table features:

- Reading + meaning display
- Kana-based index navigation
- Separate elementary / secondary sections
- Sticky navigation while scrolling
